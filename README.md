# Basira: AI-Driven Vision Assistant
**Institution:** Riphah School of Computing & Innovation (RSCI), Lahore.

## 👁️ Overview
Basira is an assistive technology solution designed to empower the visually impaired. It utilizes **YOLO 26 NVM** for real-time object detection and obstacle avoidance, integrated with a bilingual (Urdu/English) voice assistant.

## 🛠️ Tech Stack
- **Frontend:** Flutter (Dart)
- **Backend:** Node.js & Supabase (PostgreSQL)
- **Vision Engine:** YOLO 26 NVM (Native Vision Model)
- **Documentation:** IEEE Standards

## 📂 Repository Structure
- `/Documentation`: Project Report (4 Chapters) & Feature Form.
- `/Design_Models`: UML Diagrams (Use Case, ERD, Class, Sequence).
- `/Prototype`: High-fidelity UI screens and accessibility mockups.
